# FM 接管模式 — 测试运行指南

## 启动

### 一键启动（推荐）

```bash
start_fm.bat
```

### 手动启动（5 终端）

```bash
# 终端 1: Stock    python -m stock_management.main
# 终端 2: 调度      python -m scheduling.main
# 终端 3: 诊断     python -m tcp_diagnosis.main
# 终端 4: FM       python -m feeding_master.main
# 终端 5: HMI      python -m upper_hmi.main
```

## 操作流程

### 手动模式（调度服务 OFF）

```
1. HMI 启动 → 桥接+FM自动连接, 仿真开始
2. 点击画布上的料仓 → 选上料点/路线 → 手动激活
3. 路线自动: 小车移动→上料→清空→节能待机
4. 顶栏「停止」可停止正在运行的路线
5. 急停仅在手动模式可用
```

### 自动模式（调度服务 ON）

```
1. HMI 启动
2. 点击「调度服务」按钮(仅激活, 不自动调度)
3. 点击「全部自动」或单独 D6/D7/D8/D9 → 选择 D7 上料点 → FM 启动调度
4. 四条皮带自动运行: 小车移动→上料→清空→续料→循环
5. 关闭「调度服务」即关闭自动, 回到手动
```

## 常见问题

| 问题 | 排查 |
|------|------|
| 皮带不动 | 检查 FM 终端是否有调度日志, Stock 是否有料位数据 |
| 卡在清空 | 检查诊断服务是否运行, FM 终端是否有异常日志 |
| 手动停止不生效 | 顶栏「停止」按钮调用 FM 的 manual_stop |
| D8 换列误判 | 最新版已修复, 确认代码为最新 |
