# 搅拌站后料场上料系统仿真软件

基于 PyQt5 开发的搅拌站后料场上料系统仿真软件，用于测试验证控制逻辑。

## 系统组成

根据图纸，系统包括以下组成部分：

- **上料点**：物料起始位置
- **1#皮带**：水平输送段
- **提升段**：物料提升区域
- **2#皮带**：第二级水平输送
- **3#皮带**：垂直输送至中转斗
- **中转斗**：中间缓存装置
- **高位储料仓**：最终存储位置
- **接近开关传感器（S1-S5）**：检测物料位置

## 功能特性

### 仿真动画
- 皮带运行动画（方向箭头）
- 物料流动可视化
- 传感器状态显示（发光效果）
- 中转斗/储料仓料位实时显示

### 控制面板
- 皮带独立启停控制
- 仿真速度调节（0.5x - 3x）
- 手动/自动供料模式
- 系统复位
- 紧急停止

### 状态监控
- 皮带运行状态
- 传感器触发状态
- 料位百分比显示
- 物料计数统计
- 运行时间统计

### 报警管理
- 传感器超时报警
- 储料仓满报警
- 中转斗满报警
- 紧急停止报警

### 日志记录
- 传感器触发日志
- 皮带状态变更日志
- 报警事件日志
- 物料流动日志
- CSV导出功能

## 使用说明

1. **启动皮带**：点击皮带控制按钮启动1#、2#、3#皮带
2. **供料**：手动点击"供料"按钮或开启"自动模式"
3. **调整速度**：拖动速度滑块调整仿真速度
4. **监控状态**：观察右侧状态面板的实时数据
5. **查看日志**：底部日志面板记录所有事件
6. **导出数据**：点击"导出CSV"保存日志

## 快捷键

- `F5`：启动仿真
- `F6`：停止仿真
- `F12`：紧急停止
- `Ctrl+R`：复位系统
- `Ctrl+E`：导出日志
- `Ctrl+Q`：退出程序
