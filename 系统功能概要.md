## 二、核心功能模块

### 1. 多路线仿真引擎
- **8条上料路线**，覆盖4条终点皮带（D6/D7/D8/D9）
- **19条皮带** + **20个接近开关** + **7个中转斗** + **5个上料点**
- **28仓配料站**（P1-P4列 × 7行）+ **12仓高位储料仓**（S1-S6 × 2行）

### 2. 6状态路线状态机
| 状态 | 说明 | 进入条件 |
|------|------|---------|
| MOVING_TO_TARGET | 小车移动 | 路线启动/下一料仓 |
| FEEDING | 正常补料 | 小车到达目标位置 |
| CLEARING | 清空余料 | 料位达到阈值(88-98%) |
| WAITING | 等待 | 余料清空完毕 |
| STANDBY | 节能待机 | 无待补料仓 |
| IDLE | 空闲 | 路线停止 |

### 3. 3种清空策略（D7/D8独有）
- **顺序策略**（sequential）：终点皮带停，3秒后小车提前移动，非终点皮带清空
- **反序策略**（reverse）：所有皮带运行清空，小车最后移动
- **换列策略**（column_switch）：D8专属，跨列移动时中转斗保持开放

### 4. 自动上料调度系统
- TCP调度服务（端口8891-8894），独立进程运行
- **触发规则**：
  - 紧急：任意料仓 < 10% → 立即调度（120s冷却）
  - 空闲：无执行路线 & 料仓 < 70t → 调度（120s冷却）
  - 预请求：当前为序列最后仓 & 料位 ≥ 80% → 提前调度
- 调度算法：遗传算法 + 贪心启发式优化

### 5. 物料物理仿真
- 物料从**上料点**出发 → 沿**皮带**运动 → 经**中转斗**转运 → **分料小车**落料 → 到达**目标料仓**
- 物料驱动料位：到达目标仓才增加（非时间驱动）
- 中转斗囤积与释放：清空时关闭斗囤积余料，补料时0.195t/s释放
- 料仓消耗：实时消耗模拟搅拌站生产

### 6. 实时2D动画渲染
- 暗色主题工业风格界面
- 皮带运行/停止状态实时显示
- 物料粒子动画（石粉/10mm/20mm三种骨料）
- 中转斗黄色填充（实时反应囤料量，8500kg容量）
- 分料小车平滑移动动画
- FEEDING时紫色/黄色闪烁补料箭头
- 高位储料仓颜色分级（红→橙→黄→绿）

### 7. 故障诊断系统
- **接近开关故障**：卡高/卡低检测
- **中转斗故障**：开关卡死、称重异常
- **皮带故障**：速度异常检测（波动>30%）
- **小车传感器故障**：位置/极限/分料故障注入
- 本地/TCP双模式诊断

### 8. TCP/UDP下位机通信
- TCP JSON数据推送（端口8888，0.3s间隔）
- UDP 57字节二进制帧（端口8889，0.3s间隔，CRC16校验）
- 包含传感器、中转斗、皮带速度、小车位置等完整数据

## 四、解耦架构（decoupled_system）

独立的前后端分离版本，支持多前端接入：

```
┌──────────┐  ┌──────────┐
│ 上位机A  │  │ 上位机B  │  (PyQt5 / 第三方)
│ TCP客户端│  │ TCP客户端│
└────┬─────┘  └────┬─────┘
     │   TCP:9501  │
     │  JSON命令/状态│
     └──────┬──────┘
     ┌──────┴──────┐
     │  后端服务    │  (headless, 零Qt依赖)
     │  server/    │
     │  SimulationEngine
     │  TCP Server
     └─────────────┘
```
